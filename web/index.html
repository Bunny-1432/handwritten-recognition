Map-to-Sequence Layer
Reshape CNN feature maps from spatial grids to sequential features. Each column becomes a timestep, creating a natural left-to-right reading order.
  • (B, C, H, W) → (B, W, C×H)
  • Preserves spatial locality
  • Variable-width input support
🔄
Bidirectional LSTM
Two stacked BiLSTM layers with 256 hidden units capture context from both directions — essential for ambiguous characters resolved by surrounding context.
  • 2-layer, hidden_size=256
  • Forward + backward context
  • Output: per-timestep probabilities
📊
CTC Loss & Decoding
Connectionist Temporal Classification handles alignment-free training. Beam search decoding with optional language model integration for production accuracy.
  • No explicit segmentation needed
  • Blank token for spacing
  • Beam search + LM rescoring
⚡
Training Data & Transfer
Leverage the trained CNN backbone with frozen weights, then fine-tune on word-level datasets like IAM Handwriting (115K words).
  • IAM, RIMES, CVL datasets
  • Phase: freeze CNN → train LSTM → fine-tune all
  • Expected: ~8% CER on IAM
